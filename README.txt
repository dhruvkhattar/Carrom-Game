Open the game.html file in browser to start the game.

Rules:
Initial Score = 100
Every 5 secs, your score reduces by 1.
You earn 5 points if you pocket a white coin.
You lose 20 points if you pocket a black coin or the striker.
You earn 20 points if you pocket the queen(red coin).

Controls:
Left arrow  : Move Left
Right arrow : Move Right
A           : Aim Left
D	    : Aim Right
S	    : Decrease Speed
W	    : Increase Speed
Space       : Shoot

You can also change the speed by scrolling.
